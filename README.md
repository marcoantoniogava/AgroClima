# AgroClima
Sistema de monitoramento climático operacional de propriedades agrícolas.

A proposta consiste no desenvolvimento de uma nova funcionalidade denominada AgroClima, destinada ao monitoramento climático operacional de propriedades agrícolas.

O objetivo do sistema é permitir que usuários cadastrem propriedades agrícolas e acompanhem previsões meteorológicas relevantes para operações do campo, transformando dados climáticos em informações operacionais úteis, como alertas automáticos e identificação de janelas favoráveis ou desfavoráveis para atividades agrícolas.

O sistema utilizará dados meteorológicos provenientes de uma API externa (Open-Meteo), os quais serão processados pelo sistema para geração de previsões e análise de condições climáticas para atividades como pulverização, plantio e colheita.
